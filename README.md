# DevBank 💰

Sistema bancário em Java desenvolvido para praticar Programação Orientada a Objetos.

## Funcionalidades
- Criar conta corrente e poupança
- Depositar
- Sacar
- Transferir valores
- Consultar saldo
- Ver histórico
- Aplicar rendimento na poupança

## Tecnologias
- Java
- Programação Orientada a Objetos
- IntelliJ IDEA / VS Code

## Estrutura
- `Cliente`
- `Conta`
- `ContaCorrente`
- `ContaPoupanca`
- `Banco`
- `Main`

## Objetivo
Praticar conceitos como herança, encapsulamento, listas e menus interativos em Java.
